# Sky Siege Industrial

## コンセプト

**Sky Siege Industrial** は、地上が危険地帯となった世界で、空中に工業拠点・飛行船・移動要塞を築いて生き延びることを目的とした Minecraft 1.20.1 Forge 向け Modpack コンセプトです。

地上にはホード、強化 Mob、廃墟都市、ダンジョン、危険なバイオームが広がっています。プレイヤーは地上に恒久拠点を作らず、資源・戦利品・設計素材を求めて危険な地上へ降下し、回収後は空中拠点へ帰還します。

空は比較的安全ですが、資源はありません。地上は危険ですが、工業化・武装化・拠点拡張に必要なものが眠っています。

> 地上は危険。  
> 空は安全。  
> でも、冒険と資源は地上にしかない。

最終的な目標は、Create / Valkyrien Skies / Mekanism / AE2 / Immersive Engineering を使って、武装した空中工業要塞 **Sky Fortress** を完成させ、地上の危険地帯を攻略・奪還することです。

---

## 推奨環境

- Minecraft: **1.20.1**
- Mod Loader: **Forge**
- プレイ方針: 工業 + 空中拠点 + 物理演算 + 銃火器 + 高難易度サバイバル + 探索

## 主要 Mod

### コア・物理演算・空中拠点系

- **Valkyrien Skies 2**  
  物理演算付きの船・飛行船・移動構造物の基盤。

- **Eureka! Ships! for Valkyrien Skies**  
  ブロックで作った船や飛行船を比較的簡単に動かせるようにする。

- **Create**  
  機械、動力、搬送、加工ライン、ギミックの中心。

- **Clockwork**  
  Create と Valkyrien Skies を連携させる本命アドオン。Create の機械要素を使った物理演算マシン作りに使う。

- **Create: Interactive**  
  Create の contraption をより物理オブジェクト的に扱うための連携 Mod。

- **Create: VS Engineering**  
  Create + Valkyrien Skies の制御・機構系を増やすアドオン。

- **Create: Propulsion**  
  スラスター、推進、センサー系。空中工業船を作るなら相性が良い候補。

- **Trackwork**  
  履帯や車輪を追加。地上探索車両、採掘車両、戦車風マシン用。

---

### 工業系

- **Mekanism**  
  高性能な工業機械、鉱石処理、ガス処理、装備、終盤技術を担当。

- **Mekanism Generators**  
  Mekanism の発電拡張。中盤以降の発電基盤。

- **Applied Energistics 2 / AE2**  
  倉庫、自動クラフト、物流の中枢。空中拠点のメインストレージとして使う。

- **Immersive Engineering**  
  重工業、発電、ケーブル、マルチブロック機械。空中要塞や工業船の見た目と相性が良い。

- **Create Crafts & Additions**  
  Create と電力系工業 Mod の橋渡し。

- **Create Big Cannons**  
  Create 系の大砲。空中要塞の主砲や防衛設備として使う。

---

### 武器・戦争系

- **Timeless and Classics Zero / TaCZ**  
  銃火器 Mod。地上探索やダンジョン攻略のメイン武装。

- **Superb Warfare**  
  銃、大砲、車両、兵器系。地上制圧や空中拠点の武装化に使う。

- **Create Big Cannons**  
  工業と戦争の中間枠。Create で作る砲台として相性が良い。

### 難易度上昇・地上危険化系

- **The Hordes**  
  ホード襲撃や感染系の要素を追加。地上を長居したくない場所にする。

- **Born in Chaos**  
  強めの敵 Mob を追加。探索時の緊張感を上げる。

- **Majrusz's Progressive Difficulty**  
  進行に応じて難易度を上げる。序盤から理不尽になりすぎず、徐々に危険化できる。

- **Enhanced AI**  
  Mob AI を強化。

- **Zombie Awareness**  
  ゾンビが音や血、光などに反応するようになり、地上での行動に緊張感が出る。

- **Improved Mobs**  
  敵が装備を持ったり、より厄介になる。導入時は難易度調整推奨。

## 進行ルール案

### 基本ルール

- 地上に恒久拠点を作ってはいけない。
- 地上に置けるのは仮設キャンプ、採掘前哨基地、防衛塔、補給拠点のみ。
- AE2 のメインストレージは空中拠点のみ。
- Mekanism の上位機械は空中拠点のみ。
- Digital Miner は中盤後半まで禁止。
- MekaSuit は終盤まで禁止。
- Waystone は空中拠点または攻略済み拠点のみ設置可能。
- 地上での長時間放置は禁止。
- ホード発生時、地上施設は放棄してもよい。

## クエストサンプル

FTB Quests などで実装することを想定した、章ごとのクエスト案です。

---

### 第0章：墜落 / Crash Landing

目的: 最低限の生存準備を整え、地上が安全ではないことを理解する。

- 木材を入手する
- 石ツール一式を作る
- 食料を確保する
- 仮設キャンプを作る
- 初日の夜を生き延びる
- 地上に恒久拠点を作らないルールを確認する

報酬案:

- 松明
- パン
- 包帯や簡易回復アイテム
- 初期バックパック

---

### 第1章：地上サバイバル / Ground Survival

目的: 地上で最低限の工業準備を行い、空へ逃げるための素材を集める。

- 鉄を入手する
- 銅を入手する
- Create の Andesite Alloy を作る
- Create の水車または風車を作る
- Immersive Engineering の Engineer's Hammer を作る
- Coke Oven を作る
- Treated Wood を作る
- 地上の廃墟またはダンジョンを1か所探索する

報酬案:

- Create 用素材
- IE 用素材
- 弾薬少量
- 地図またはコンパス

---

### 第2章：初期工業化 / First Industry

---

### 第3章：初飛行 / First Flight

- 建材

- 修理素材
- 地上探索用の弾薬

---

### 第4章：空中拠点化 / Sky Base

目的: 空中で生活・工業・保管ができる状態を作る。

- 空中拠点に農業区画を作る
- 空中拠点に発電区画を作る
- 空中拠点に加工区画を作る

- AE2 の ME Chest を作る

報酬案:

- AE2 素材
- 建築ブロック
- 食料生産用アイテム

---

### 第5章：武装降下作戦 / Armed Expedition

目的: 地上探索を本格化し、危険地帯から戦利品を持ち帰る。

- TaCZ の銃を入手または作成する
- 弾薬を生産する
- Superb Warfare の基本武器を作る

- AE2 Processor 素材

- 防衛設備素材

### 第8章：地上奪還 / Reclaim the Surface

目的: 空中要塞の力で、危険な地上を攻略・制圧する。

- 大型ダンジョンを3つ攻略する
- 廃墟都市を1つ制圧する
- 地上に防衛済み採掘前哨基地を3つ作る
- Mekanism の核分裂炉を稼働させる

- 最終遠征用の補給物資を準備する
- 地上に記念碑またはビーコンを建てる

報酬案:

- クリア記念アイテム
- 装飾ブロック
- 終盤装備素材

## メモ

この Modpack は、単に工業を進めるだけでなく、以下のループを重視します。

1. 空中拠点で準備する
2. 地上へ降下する
3. 資源・戦利品・設計素材を回収する
4. 危険になる前に撤退する
5. 空中拠点を拡張する
6. より危険な地上へ挑む

工業化は目的ではなく、地上を攻略するための手段です。
